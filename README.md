# ModManager
> A Mod manager for SecretFlaserManaka.
- (Now it's only a LOCAL mod manager)

# Install
Tauri + React + Typescript. Need Node.js and Rust. See [Tauri pre-requisites](https://tauri.app/start/prerequisites/)

```powershell
cd ModManager

# Install
npm install

# Debug
npm run tauri dev

# Release
npm run tauri build
```
